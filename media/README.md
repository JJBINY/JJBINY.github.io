# Media Source

영상 원본과 썸네일 이미지는 여기에 둡니다.

추천 구조:
- `steel-demo.mp4`
- `indexing-demo.mp4`
- `query-demo.mp4`
- `steel-poster.jpg`
- `indexing-poster.jpg`
- `query-poster.jpg`

GitHub Pages에서는 `mp4`, `webm`, `jpg`, `png` 파일을 정적으로 바로 서빙할 수 있습니다.
